Step 1: Upload JavaScript File

  1. Download knowledge-center-shopify.js
  2. In Shopify Admin: Online Store ‚Üí Themes ‚Üí Edit code
  3. In the Assets folder, click Add a new asset
  4. Upload the file

Step 2: Create Section

  1. In the Sections folder, click Add a new section
  2. Name it knowledge-center
  3. Copy the content from knowledge-center-section.liquid
  4. Save

Step 3: Add Articles

  1. Create articles in this admin tool
  2. Click Export JSON for Shopify
  3. Copy the JSON output
  4. In Shopify theme customizer, add the Knowledge Center section to a page
  5. Paste the JSON into the Articles JSON Data field
  6. Customize colors and settings
  7. Save

Step 4: Configure User Tags

To use gated content (like affiliate-only articles):

  1. In Shopify Admin: Customers
  2. Select a customer
  3. Add tags like: affiliates, vip, premium
  4. Save customer

The section automatically checks customer tags and shows/hides content accordingly.
